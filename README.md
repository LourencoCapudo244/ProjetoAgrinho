# 🌾 EcoDash - Simulador de Pegada de Carbono

Projeto tecnológico desenvolvido para o **Concurso Agrinho 2026**.

## 🚀 Objetivo
O EcoDash é um simulador web que calcula a pegada de carbono residencial, conscientizando a comunidade sobre o impacto do consumo diário e promovendo o tema do ano: *"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"*.
